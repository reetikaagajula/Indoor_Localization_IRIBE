# Indoor Localization — Iribe Center Floor 5
### University of Maryland, College Park

WiFi-based indoor localization system for the Brendan Iribe Center (Floor 5) using RSS fingerprinting and RTT ranging. Combines KNN, HORUS, and RTT trilateration to estimate user position on the floor plan.

---

## Results

| Method | Mean Error | Median Error |
|---|---|---|
| KNN (RSS only) | 3.90m | 3.19m |
| HORUS (RSS only) | 6.59m | 5.71m |
| RTT only | 6.42m | 5.80m |
| **Combined RSS + RTT** | **3.69m** | **2.57m** |

Evaluated on 10 test landmarks using Leave-One-Out Cross Validation across 103 Reference Points.

## System Architecture

```
Data Collection (Android App)
        ↓
RSS Scans (WifiAnalyzer)    +    RTT+RSS Scans (Custom App)
        ↓                               ↓
Fingerprint Database              RTT Trilateration
(103 RPs × 88 APs)                (distance to APs)
        ↓                               ↓
   KNN / HORUS                   Coordinate Transform
        ↓                               ↓
        └──────── Combined Estimate ────┘
                        ↓
              Position on Floor Plan
```

## Data Collection

### Reference Points (103 RPs)
- Collected using **WifiAnalyzer** (open source Android app)
- 105 locations marked on Floor 5 hallways
- ~5 scans per location
- Files: `RP_501.csv` to `RP_5103.csv`
- **88 unique Iribe APs** identified (filtered to `50:xx` and `98:xx` prefixes)

### Test Landmarks (10 locations)
- Collected using custom Android app with **WiFi RTT (IEEE 802.11mc)** support
- Each file contains both RSS (RSSI) and RTT (distance in mm) measurements
- RTT-capable APs identified by BSSID ending in `:3`

### Coordinate System
- Floor plan coordinates derived from clicking on the digital floor plan
- Click script: `x = pixel_x / 10`, `y = (height - pixel_y) / 10`
- Scale: **1 floor unit = 0.26 meters** (floor plan scale: 1" = 26')
- X range: 37–310, Y range: 30–168

---

## Algorithms

### KNN (K-Nearest Neighbors)
- Compare live scan against all 103 RPs using Euclidean distance in signal space
- Weighted average of K nearest RPs (weight = 1/distance)
- Best K = 2 (mean error 3.80m)

### HORUS (Probabilistic)
- Gaussian probability model per AP per RP (mean + variance)
- Log-likelihood computed using only detected APs
- Weighted average of K most probable RPs

### RTT Trilateration
- Uses IEEE 802.11mc Fine Time Measurement distances
- Weighted Least Squares trilateration across all RTT-capable APs
- Coordinate transform calibrated using 2 anchor points:
  - q1 = RP_501: `floor_x = 0.6287 * rtt_x + 367.51`
  - q1 = RP_501: `floor_y = -0.0081 * rtt_y + 65.63`

### Combined RSS + RTT
- `combined = 0.6 × KNN + 0.4 × RTT`
- Falls back to KNN only if RTT unavailable
